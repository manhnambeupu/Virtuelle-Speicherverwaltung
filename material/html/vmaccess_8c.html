◆ g_count

int g_count = 0
static

global acces counter as quasi-timestamp - will be increment by each memory access

The progression of time is simulated by the counter g_count, which is incremented by vmaccess on each memory access. The memory manager will be informed by a command, whenever a fixed period of time has passed. Hence the memory manager must be informed, whenever g_count % TIME_WINDOW == 0. Based on this information, memory manager will update aging information
